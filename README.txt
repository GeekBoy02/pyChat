Chat commands:

!class <classname>: Set the user's class to the specified classname. Valid classnames are "Soldier", "Engineer", and "Explorer".
!f <enemy level> or !fight <enemy level>: Start a battle against an enemy with the specified enemy level.
!aa <speed> <intellect> <luck> or !allocate attributes <speed> <intellect> <luck>: Allocate the user's free attribute points to increase their speed, intellect, and luck by the specified amounts.
!a or !attributes: Display the user's current attributes (speed, intellect, luck).
!users: List all connected users.
